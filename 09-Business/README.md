## Decisiones tomadas (founder + agent · 2026-04-25)

Reflejadas en todos los docs nuevos:

- Free para fans en MVP con donación opcional (Ko-fi).
- Monetización empieza con artistas/managers/venues/labels — los que mueven dinero.
- Precios bajos: $5/mes o $12/año como baseline, subiendo según tier.
- Comisiones (merch, tickets) = post-tracción, no MVP.
- Capital = cuando aparezca oportunidad, no forzado.
- Artist Pro **SÍ cobra** ($8/mes) — decisión resuelta: "artista free siempre" aplica a claim + verify + edit básico; Pro es add-on opcional. Disconsistencia founder documentada y resuelta.

## Reglas operativas

- Cada feature de Product Architecture pasa por Business & Legal para evaluar potencial de monetización (workflow en `08-Legal-Compliance/feature-review-workflow.md`).
- Pricing concreto se determina con **benchmarks externos** — nunca inventado.
- Partnerships requieren términos escritos. Agent drafts, founder firma.
- Revenue tracking: single source of truth = Stripe dashboard + Ko-fi dashboard + monthly snapshot.
- Stakeholder outreach se coordina con Community & Outreach — agent define estrategia + materials, Community ejecuta outreach masivo.
