## Contexto

Este doc mantiene las keywords prioritarias en español para Dauton Media. Cada keyword se vincula con una feature o página que la captura.

Público primario: **diáspora venezolana** (Miami, España, Colombia, USA latino). Keywords en ES dominan.

---

## Framework

Agrupar por intent:

1. **Search por artista** — "cuánto gana [artista]", "discografía de [artista]", "biografía de [artista]".
2. **Search por tema** — "rap venezolano", "artistas de Caracas", "beef de rap latino".
3. **Search por rol** — "mejores productores de rap latino", "labels venezolanas".
4. **Search cultural/histórico** — "historia del rap venezolano", "timeline del hip hop latino".

Cada keyword se marca con:
- Volumen estimado ES (HIGH / MED / LOW)
- Dificultad SEO (HIGH / MED / LOW)
- Intent (informational / navigational / transactional)
- Página/feature que la captura
- Prioridad (P0/P1/P2)
